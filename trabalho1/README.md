# Trabalho 1 - Space Invaders

Este projeto utiliza as libs estáticas `pthreads` e `ncurses`.

Para compilar:

```bash
# "make clean" caso queira limpar o projeto dos artefatos de uma compilação anterior;
make
```

Para executar:

```bash
./bin/game
```
